# Quick Start for AI Consumers

Ready to start using AI services? This guide will help you get up and running quickly.

## Overview

As an AI consumer, you can interact with services on the Decentralized AI Platform in several ways:

1. **[Marketplace UI](./calling-via-marketplace/)** - Best for beginners
2. **[Command Line (CLI)](./calling-via-cli/)** - For developers comfortable with terminal
3. **[Terminal UI (TUI)](./calling-via-tui/)** - Interactive terminal interface
4. **[SDK Integration](./calling-via-sdk/)** - For application developers
5. **[Service Training](./service-training/)** - Fine-tune AI models with your data

---

## Quick Start Path

### Step 1: Get Ready (5-10 minutes)

[Complete the Getting Ready Checklist](./getting-ready-to-call-ai/)

- Create an Ethereum wallet
- Get ETH for gas fees  
- Acquire ASI (FET) tokens
- Create marketplace account
- (Optional) Set up development environment

### Step 2: Choose Your Interface

Pick the method that best suits your needs:

#### Marketplace UI (Recommended for beginners)
**Best for**: First-time users, testing services, visual interface
**Setup time**: 5 minutes
**Technical level**: No coding required

[Start with Marketplace](./calling-via-marketplace/)

#### Command Line Interface (CLI)
**Best for**: Automation, scripting, power users
**Setup time**: 10-15 minutes
**Technical level**: Basic terminal knowledge

[Start with CLI](./calling-via-cli/)

#### Terminal User Interface (TUI)
**Best for**: Interactive terminal users, service management
**Setup time**: 10-15 minutes
**Technical level**: Comfortable with terminals

[Start with TUI](./calling-via-tui/)

#### SDK Integration
**Best for**: Developers building applications
**Setup time**: 15-30 minutes
**Technical level**: Programming experience (Python/JavaScript)

[Start with SDK](./calling-via-sdk/)

### Step 3: Call Your First Service

Follow the guide for your chosen method to:
1. Connect your wallet
2. Deposit funds
3. Open a payment channel
4. Make your first service call

### Step 4: Explore Advanced Features

Once comfortable with basic calls:
- [Train custom models](./service-training/) with your data
- Set up automated workflows
- Integrate services into your applications

---

## Quick Reference

### Payment Flow

```
1. Deposit ASI → Escrow Account
2. Open Channel → Allocate funds to service
3. Call Service → Deduct from channel
4. (Optional) Add funds or extend channel
```

### Cost Structure

- **One-time**: Gas fees for deposits and channel operations
- **Per-call**: Service-specific ASI token cost
- **No additional fees**: Once channel is open, calls are gasless

### Free Trials

Many services offer free trial calls:
- No payment channel required
- Limited calls per user
- Perfect for testing
